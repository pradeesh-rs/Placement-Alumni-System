# Placement & Alumni Management System

A web-based Placement and Alumni Management System developed using **Python (Django)** to simplify placement activities, student management, company recruitment, and alumni tracking for educational institutions.

---

## Features

### Student Module
- Student Registration & Login
- Profile Management
- Resume Upload
- CGPA Calculator
- View Eligible Jobs
- Apply for Jobs
- Training Notifications
- simple chatbot

### Teacher Module
- Student Approval
- Resume Verification
- Generate Reports
- Manage Training Programs
- Alumni Management

### Company Module
- Company Registration
- Post Job Opportunities
- View Eligible Students
- Manage Applications

### Admin Module
- Dashboard
- User Management
- Placement Reports
- Notification Management

## Installation

Clone the repository

```bash
git clone https://github.com/pradeesh-rs/Placement-Alumni-System.git
```

Move into the project directory

```bash
cd Placement-Alumni-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run migrations

```bash
python manage.py migrate
```

Start the server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000
```
